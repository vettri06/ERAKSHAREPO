# IoT Security Scanner

## Project info

This is a comprehensive IoT Security Scanner application built with React, Vite, and Python backend.

## Getting Started

### Prerequisites

- Node.js & npm
- Python 3.8+
- Nmap (installed on the system)

### Installation

1. Clone the repository
2. Install frontend dependencies:
   ```sh
   cd frontend
   npm install
   ```
3. Install backend dependencies:
   ```sh
   cd backend
   pip install -r requirements.txt
   ```

### Running the Application

1. Start the backend server:
   ```sh
   cd backend
   python api.py
   ```

2. Start the frontend development server:
   ```sh
   cd frontend
   npm run dev
   ```

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Python (FastAPI/Flask)
- Nmap
